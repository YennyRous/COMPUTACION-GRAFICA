MALLA POLIGONAL ESFÉRICA

Explicación del código

Librería utilizada:
Se utiliza Three.js, una librería de JavaScript que permite crear gráficos 3D en el navegador mediante WebGL.

1. Escena:
THREE.Scene() crea el espacio donde se renderizan los objetos.

2. Cámara:
PerspectiveCamera define la perspectiva desde donde se visualiza la escena 3D.

3. Renderizador:
WebGLRenderer se encarga de dibujar los gráficos en el navegador.

4. Geometría:
SphereGeometry crea una esfera usando segmentos horizontales y verticales.

5. Wireframe:
WireframeGeometry convierte la esfera en una malla de líneas (estructura poligonal).

6. Material:
LineBasicMaterial define el color y apariencia de las líneas.

7. Objeto 3D:
LineSegments une la geometría y el material para formar la esfera visible.

8. Animación:
Se usa requestAnimationFrame() para rotar la esfera continuamente.

9. Responsive:
Se ajusta automáticamente al tamaño de la pantalla cuando el usuario cambia el tamaño de la ventana.
